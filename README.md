# PROMPT – Uma Porta Se Abriu

Landing page promocional para o livro **PROMPT – Uma Porta Se Abriu** (Crônicas do Limiar, Livro 1), de Marcone Arruda.

## 🚀 Tecnologias

- HTML5 semântico
- CSS3 modular com variáveis (Design System)
- JavaScript Vanilla (sem frameworks)
- PWA (manifest.json + Service Worker)
- Static Forms para captura de leads
- GitHub Pages (deploy recomendado)

## 🔧 Configuração

1. Substitua as imagens em `assets/images/` pelas versões reais.
2. Verifique os ícones em `assets/icons/` (192×192 e 512×512).
3. Os links sociais já apontam para:
   - WhatsApp: `https://wa.link/zy3h8a`
   - Instagram: `https://www.instagram.com/marconearruda.autor/#`
4. O formulário usa a API do [Static Forms](https://staticforms.dev/) com a chave fornecida.

## 🌐 Deploy (GitHub Pages)

1. Crie um repositório no GitHub.
2. Envie todos os arquivos.
3. Ative o GitHub Pages nas configurações do repositório (branch `main`, pasta `/`).
4. Acesse `https://seuusuario.github.io/seurepositorio/`.

## 🎨 Personalização

- Cores: edite `css/variables.css`
- Fontes: Google Fonts em `index.html` e `variables.css`
- Conteúdo: edite o HTML das seções em `index.html`

## 📚 Sobre o Livro

*PROMPT – Uma Porta Se Abriu* é o primeiro volume da série **Crônicas do Limiar**. Acompanhe Rick Almeida em uma jornada por portais dimensionais, viagem no tempo e uma batalha contra os Colecionadores, uma organização que quer controlar o próprio tecido da realidade.
